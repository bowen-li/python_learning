# 列表

## 列表是什么？

一系列按特定顺序排列的元素组成。

元素：字母，数字。

元素之间可以没有任何关系。有关系的又当如何呢？

列表的名字通常可以用复数的形式，包含不止一个的元素，比较合理。

## python中的列表是什么？

\[ \]

用逗号分开元素即可。

```
persons=['Jim', 'Lucy', 'Kate']
print(persons)
```

## 怎么应用python列表？

访问：跟数组是一个原理。

          索引从0开始。与列表操作的底层实现有关。

          最后一个元素索引-1。-2是倒数第二个元素。以此类推，这个有点意思！

```
persons[0]
persons[1]
persons[-1]
```
